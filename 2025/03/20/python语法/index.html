2. 列表的常见操作和方法

2.1 添加元素

  • append()
    在列表末尾添加单个元素。

  • extend()
    将一个可迭代对象中的多个元素添加到列表末尾。

  • insert()
    在指定位置插入元素，其他元素自动右移。

2.2 删除元素

  • remove()
    删除列表中第一次出现的指定值。

  • pop()
    删除并返回指定索引处的元素（默认删除最后一个）。

  • clear()
    清空整个列表。

2.3 查找与统计

  • index()
    返回指定值第一次出现的索引位置。

  • count()
    统计某个元素在列表中出现的次数。

2.4 排序与反转

  • sort()
    就地对列表进行排序（默认升序），也可以通过参数设置排序方式。

  • sorted()
    返回一个新的排序后的列表，不改变原列表。

  • reverse()
    就地将列表中的元素反转。

2.5 列表推导式

列表推导式提供了一种简洁的方法来创建新列表。

3. 其他

  • copy()
    创建列表的浅拷贝。

  • len()
    返回列表的长度（元素个数）。
